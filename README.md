QrobotCompetition
=================

创意描述

手语是语言残障人士最重要的交流语言，然而，普通人中能看懂手语的人却非常少。因此，基于Qrobot平台，我们致力于手语识别系统的研究，设计并开发出更加灵活的人机交互界面，方便语言残障人士和普通人之间的沟通交流。
其次，手势作为一种新型的输入途径，运用到人机交互的过程中。人机之间的通讯不再需要中间媒体，用户可以通过定义一种适当的手势来对周围的机器进行控制。

应用简介

手语小Q是一款由CodeFuture团队设计和开发的实现了语言残障人士和普通人之间沟通交流的社交类应用。
应用以Qrobot为平台，采用HOG和SVM算法以及OpenCV函数库实现了动态和静态的手势识别；建立了手势库并且训练了多种手势的分类器；实现了手势与语音之间的转换，支持用户自主扩充手势库，具有比较广泛的应用前景和一定的实际价值。
